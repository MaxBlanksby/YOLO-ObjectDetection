# Sample Images

To test the Data Labeler interface, add image files (.jpg, .jpeg, .png, .bmp, .tiff, .gif) to this folder.

The interface will allow you to:
1. Load images from this folder
2. Create custom labels
3. Draw bounding boxes on objects
4. Save annotations in YOLO format

Saved annotations will be stored in the `Labeled_Data` folder.
